# FILLETIA
Filletia merupakan produk inovasi ikan fillet dan menyediakan berbagai varian bumbu rempah siap masak yaitu seperti bumbu marinasi, bumbu kuah kuning, bumbu pepes dan bumbu bakar sebagai solusi masyarakat dalam memenuhi kebutuhan dan memudahkan konsumen dalam memenuhi kebutuhan protein berupa produk olahan ikan yang praktis.
